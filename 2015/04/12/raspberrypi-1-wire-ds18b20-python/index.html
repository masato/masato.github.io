1-Wireのセットアップ

カーネルのバージョンを確認します。

1-Wire in Parasite Power configuration (1-Wire using 2 wires) does not work in 3.18.3 #348にあるように、カーネルが3.18.1+以上の場合は1-Wireを有効にする設定が変更になっています。/boot/config.txtに以下の行を追加して再起動します。
