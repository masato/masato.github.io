Reactの特徴

Reactのページに書いてある特徴を簡単にまとめます。

  • JUST THE UI
    MVCのVだけComponentとして提供する

  • VIRTUAL DOM
    VirtualDOMを使い差分だけ更新するため高速に動作する

  • DATA FLOW
    一方向のリアクティブなデータバインディング

  • JSX
    JavaScript内にXML風に記述でき、DOM構造がわかりやすい

開発用コンテナ
